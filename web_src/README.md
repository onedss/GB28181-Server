# livecms-web-src

OneGBS 前端工程源码

# 起步

```
# 准备工具
npm install -g cross-env

cd web_src
#  安装 npm 依赖
cnpm install

# 编辑 webpack.config.js > devServer > proxy target ip 指向 server ip

# 启动
npm run start
# 编译
npm run build
```

# 调主题色

1. web_src > assets > styles > element-custom.scss > $--color-primary

2. web_src > assets > styles > variables.less > @base
